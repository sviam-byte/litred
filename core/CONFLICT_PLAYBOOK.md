# CONFLICT_PLAYBOOK

## Goal
Разрешать противоречия между модулями по явным правилам.

## Primary Rules
1. corpus-limit beats all
2. structural beats semantic/style
3. epistemic warning can downgrade rhetorical praise
4. false_positive_guard can lower confidence, but not erase a well-evidenced structural defect
5. one module cannot promote taste-only preference to critical issue

## Examples
### Example 1
ai_detection: high suspicion
false_positive_guard: deliberate minimalist style
Resolution:
- keep marker
- reduce certainty
- report as contested suspicion, not verdict

### Example 2
language: repetitive diction
fiction: repetition functions as motif
Resolution:
- reclassify as mixed observation
- do not score as defect unless repetition becomes structurally flattening
