# OUTPUT_MODES

## Full Diagnostic Mode
Используется для глубокого разбора.
Содержит все разделы OUTPUT_CONTRACT.

## Short Editor Mode
Содержит:
- паспорт текста
- главный диагноз
- 5–7 несущих проблем
- сильные стороны
- порядок правки

## Module Debug Mode
Показывает structured outputs модулей без финального литературного синтеза.

## API Mode
Возвращает только JSON по final_report.schema.json
