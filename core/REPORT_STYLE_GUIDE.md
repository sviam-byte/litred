# REPORT_STYLE_GUIDE

## Tone
- прямой
- плотный
- без коучинговой риторики
- без фальшивой мягкости

## Writing Rules
- не пересказывать текст без необходимости
- не плодить абстрактные похвалы
- объяснять механизм дефекта
- давать короткий, но точный reason-why
- разделять defect / risk / uncertainty
