# WHAT_ON_OUTPUT

Это пакет о том, что агент обязан выдавать на выходе.

## General Verdict Rules (оба режима)
- Стоит вторым — всегда.
- Написан прозой — не списком.
- 5–10 предложений.
- Отвечает на: жив ли текст, где проблема, что делать.
- Не является резюме следующих разделов.
- Не является комплиментом.


## Минимальные output surfaces
1. Полный редакторский отчёт
2. Краткий редакторский отчёт
3. JSON final_report
4. JSON short_report
5. План ревизии как отдельный объект
6. Debug dump модулей

## Что обязательно должно быть на выходе
- паспорт текста
- ограничения вывода
- главный диагноз
- разбор по релевантной ветке
- языковой блок
- блок фразы и абзаца
- strengths
- cautions
- приоритеты правки
- revision plan
