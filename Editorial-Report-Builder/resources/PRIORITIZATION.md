# PRIORITIZATION

## Order of Repair
1. corpus / scope limitations
2. structural failures
3. semantic / epistemic failures
4. scene / paragraph operations
5. sentence-level mechanics
6. language / style
7. copy / proof

## Rewrite Buckets
- rewrite now
- structural next
- line edit later
- cosmetic last

## Hard Rule
Нельзя рекомендовать активный line edit, если у текста провален тезис, маршрут, сцена или арка.
